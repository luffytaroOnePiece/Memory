━━━ CRITICAL PERFORMANCE RULES ━━━
These pages load inside an iframe. Performance is non-negotiable.

ABSOLUTELY BANNED:
✘ filter: blur() on ANY element
✘ backdrop-filter on ANY element
✘ <canvas> elements
✘ requestAnimationFrame loops
✘ Infinite CSS animations (zero allowed)
✘ Animated backgrounds of any kind
✘ will-change on anything
✘ SVG filters on large elements

ALLOWED:
✓ One-shot entrance animations (opacity + transform, runs once)
✓ Hover transitions on border-color, background-color, transform
✓ CSS transitions on max-height for collapsible sections
✓ IntersectionObserver for fade-in on scroll (transform + opacity only)
✓ Smooth scroll-behavior on html
