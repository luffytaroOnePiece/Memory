━━━ BACKGROUND ATMOSPHERE ━━━
Layer these (do NOT use plain solid backgrounds):
1. Base radial/mesh gradient using 2–3 colors from --bg-base palette
2. Subtle SVG texture OR CSS noise grain (via inline SVG filter or 
   pseudo-element with repeating pattern) — set OPACITY LOW (0.03–0.06)
3. Optional: faint geometric lines, dot grid, or diagonal rule pattern
4. Vignette overlay (radial gradient, dark edges)

━━━ CRITICAL PERFORMANCE RULES ━━━
These pages load inside an iframe. Performance is non-negotiable.

ABSOLUTELY BANNED:
✘ filter: blur() on ANY element (causes massive GPU compositing)
✘ backdrop-filter on ANY element (forces per-frame background blur)
✘ Animated elements larger than 200px with filter/blur
✘ <canvas> with requestAnimationFrame for background effects
✘ More than 2 infinite CSS animations on the entire page
✘ will-change on elements that never animate
✘ SVG filters applied to large/full-page elements

ALLOWED ANIMATIONS (lightweight):
✓ Entrance animations using transform + opacity (run once, not infinite)
✓ Staggered entrance delays via nth-child (animation-fill-mode: both)
✓ Hover transitions on transform, opacity, box-shadow, border-color
✓ ONE small decorative infinite animation (e.g. a subtle pulse on a badge)
✓ CSS transitions on max-height for reveal panels

BACKGROUND RULES:
✓ Use CSS gradients (radial-gradient, linear-gradient) — they are FREE
✓ SVG noise textures as background-image are fine (static, not animated)
✓ Keep noise/grain overlays at opacity 0.03–0.06 max
✘ Never animate background-position or background-size continuously

[MAIN TABLE / CARD GRID]

· Category pills — color-coded with tinted border + bg (NEVER solid fill)

· Difficulty as 5 pip dots (CSS circles: filled vs unfilled)
· Tier/Score badge — large styled letter: S+ > S > A+ > A > B+ > B > C

Top 3 rows:

  - Subtle metallic background tint
  - Rank badge glow (box-shadow only, no filter)

Hover state (CSS-only):

  - Detail panel shows: lore blurb, bonus stat, or contextual tip
