━━━ BACKGROUND ATMOSPHERE ━━━
Layer these (do NOT use plain solid backgrounds):
1. Base radial/mesh gradient
2. Subtle SVG texture OR CSS noise grain (via filter: url(#noise) or 
   pseudo-element with repeating pattern)
3. Optional: faint geometric lines, dot grid, or diagonal rule pattern
4. Vignette overlay (radial gradient, dark edges)

━━━ PERFORMANCE RULES ━━━
- Use will-change: transform only on animated elements (remove after animation)
- CSS animations use transform + opacity ONLY (no layout-triggering props)
- Use animation-fill-mode: both on all entrance animations
- Stagger delays: nth-child multiplied offset (e.g. calc(0.08s * var(--i)))
- Set --i via inline style="--i:1" on each row/card for clean stagger
- Intersection Observer for scroll-triggered reveals (not scroll events)
- Hover panels use CSS clip-path or max-height transitions (no JS toggling)
- No layout thrash: avoid animating width, height, top, left, margin, padding
